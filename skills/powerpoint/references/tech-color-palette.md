# Tech/Business Color Palette — Proven Combination

Used successfully in AI Agent Trends 2025-2026 presentation. Dark navy + cyan accent with white card components.

## Core Colors

| Role | Color | Hex | Usage |
|------|-------|-----|-------|
| Primary BG (dark) | Dark Navy | `#0A1628` | Cover, closing slides |
| Primary BG (light) | White | `#FFFFFF` | Content slides |
| Accent | Cyan | `#00B4D8` | Highlights, bars, active elements |
| Accent Light | Light Cyan | `#48CAE4` | Secondary highlights |
| Accent Green | Teal | `#38B2AC` | Success, positive metrics |
| Accent Orange | Orange | `#ED8936` | Warning, attention |
| Accent Purple | Purple | `#805AD5` | Tertiary accent |
| Accent Red | Red | `#E53E3E` | Challenges, risks |
| Text Primary | Near Black | `#1A202C` | Headings on white |
| Text Secondary | Gray | `#4A5568` | Body text |
| Text Muted | Light Gray | `#718096` | Captions, sources |
| Card BG | Off White | `#F0F4F8` | Bottom bars, badges |
| Card Shadow | — | `0 2px 12px rgba(0,0,0,0.08)` | Subtle elevation |

## CSS Patterns

```css
.accent-bar { position: absolute; top: 0; left: 0; width: 100%; height: 4px; background: #00B4D8; }
.stat-card { background: #fff; border-radius: 10px; box-shadow: 0 2px 12px rgba(0,0,0,0.08); padding: 20px; text-align: center; }
.stat-num { font-size: 38px; font-weight: 800; }
.stat-label { font-size: 13px; color: #1A202C; font-weight: 700; margin-top: 8px; }
.bottom-bar { position: absolute; bottom: 18px; left: 40px; right: 40px; background: #F0F4F8; border-radius: 8px; padding: 10px 18px; font-size: 13px; }
.card { background: #fff; border-radius: 10px; box-shadow: 0 2px 12px rgba(0,0,0,0.08); }
```

## Typography

- Font stack: `-apple-system, "PingFang SC", "Microsoft YaHei", "Helvetica Neue", Arial, sans-serif`
- Slide title: 32-36px, weight 800
- Card title: 16-17px, weight 700
- Body text: 12-13px
- Caption/source: 10-11px, color `#718096`

## When to Use

- Data-heavy reports, market analysis, tech trends
- Professional/consulting presentations
- Slides with stat cards, grids, timelines, comparison layouts
