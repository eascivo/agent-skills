# Apple / Tesla Keynote Dark Style

**用户明确偏好**：苹果发布会、特斯拉 AI Day 级别的暗色电影感简约。不是"少"，是**克制但有冲击力**。

## 核心特征

- **纯黑背景** `#000000`，不是深灰不是渐变
- **超大字号**：关键数据 60-96pt，标题 36-72pt
- **一页一个核心信息**（One Idea Per Slide — Apple 设计师原话）
- **电影感留白**：≥40% 空白区域
- **Helvetica Neue** 字体族（苹果系统字体）
- **单一强调色**，不堆砌多色

## 配色系统

| 角色 | 色值 | 用途 |
|------|------|------|
| 背景 | `000000` | 纯黑 |
| 主文字 | `FFFFFF` | 标题、核心内容 |
| 灰文字 | `86868B` | 副标题、描述（Apple gray） |
| 暗灰文字 | `48484A` | 极淡标注、英文标签 |
| 强调蓝 | `2997FF` | 数字、标签、线条（Apple blue） |
| 卡片背景 | `1C1C1E` | 深色卡片/面板 |
| 浅卡片 | `2C2C2E` | 次级卡片 |
| 分割线 | `38383A` | 极淡水平线 |

**辅助强调色**（仅用于数据区分，每色一个维度）：

| 颜色 | 色值 | 用途示例 |
|------|------|----------|
| 红 | `FF453A` | 长视频平台 |
| 橙 | `FF9F0A` | 图文平台 |
| 绿 | `30D158` | 播客平台 |
| 紫 | `BF5AF2` | 知识付费 |
| 金 | `FFD60A` | Newsletter |

## 字号规范

| 元素 | 字号 | 字重 |
|------|------|------|
| 封面标题 | 64-72pt | Bold |
| 超大数据 | 96pt | Bold |
| 章节标题 | 32-44pt | Bold |
| 大陈述句 | 44-48pt | Bold |
| 副标题 | 18-22pt | Regular |
| 正文描述 | 12-14pt | Regular |
| 英文标签 | 10pt | Bold + charSpacing:3-6 |
| 页码 | 9pt | Regular |

## 页面类型模板

### 1. 封面页
- 左对齐标题（不居中）
- 蓝色竖线（0.06"宽）在标题左侧作点缀
- 底部英文标签 + 右下角页码
- 大面积空白

### 2. 数据冲击页
- 超大数字（96pt）+ 单位（48pt）
- 下方一行描述（18pt 灰色）
- 底部分割线 + 补充数据

### 3. 大陈述页
- 居中或左对超大文字（44-68pt）
- 英文回声标签（12pt 极淡）
- 一句话引言（italic）

### 4. 卡片矩阵页
- 深色圆角卡片（`1C1C1E`，rectRadius: 0.12）
- 每张卡：彩色圆点 + 标题 + 描述 + 彩色数据
- 2×3 或 2×2 网格

### 5. 流程/步骤页
- 水平编号圆圈（蓝边框 + 编号）
- 圆圈间细灰连接线
- 每步：标题（18pt bold）+ 描述（10pt）

### 6. 数据条页
- 左侧百分比（16pt 蓝色 bold）
- 半透明条形图（transparency: 70）
- 右侧标签 + 小字描述

### 7. 统计列页
- 左侧蓝色竖线（0.04"宽）
- 大数字（42pt）+ 标签（14pt）+ 补充（10pt）
- 三列并排

### 8. CTA 收尾页
- 居中大字（64pt）
- 英文回声 + 蓝色短横线
- 金句（italic）

## 设计禁忌

- ❌ 渐变背景（纯黑就对了）
- ❌ 多色混搭（一个强调色为主）
- ❌ 居中段落文字（只居中标题和单行句）
- ❌ 标题下方装饰线（AI 生成标志）
- ❌ 毛玻璃/暗色科技风（用户明确反感）
- ❌ 密集卡片排列（留够间距）
