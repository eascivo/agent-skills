# 简约设计参考 — 大师、原则与配色

## 一、核心设计大师

### 1. Dieter Rams（迪特·拉姆斯）— "少，但更好"
- **身份**：德国工业设计师，Braun 首席设计官（1955-1995）
- **核心理念**："Less, but better" — 回归纯粹，回归简洁
- **十大设计原则**（直接影响 Apple）：
  1. **好的设计是创新的** — 创新的可能性永无止境
  2. **好的设计是实用的** — 强调有用性，排除干扰
  3. **好的设计是美观的** — 产品的美感是其实用性的组成部分
  4. **好的设计让产品易于理解** — 直觉化操作
  5. **好的设计是克制的** — 不抢戏，服务于内容
  6. **好的设计是诚实的** — 不夸大，不误导
  7. **好的设计是持久的** — 避免时髦元素
  8. **好的设计是细致入微的** — 没有任何细节是随意的
  9. **好的设计是环保的** — 资源节约
  10. **好的设计是尽可能少的设计** — 少即是多，回归纯粹
- **PPT 应用**：每页只保留一个核心信息；删除一切装饰性元素；留白即呼吸
- **代表色**：白、灰、黑 + Braun 标志性的米白/浅棕

### 2. Massimo Vignelli（马西莫·维涅利）— 现代主义图腾
- **身份**：意大利裔美国设计师，Helvetica 代言人，NYC 地铁图设计者
- **核心理念**：只需 5 种字体（Helvetica、Futura、Garamond、Bodoni、Times New Roman）
- **关键名言**："The grid is the underwear of the book"（网格是书的内衣）
- **设计哲学**：
  - 信息架构师 — 将复杂信息压缩为可消化的视觉系统
  - 几何形式 + 大面积留白
  - 严格的网格系统
  - 色彩克制，偏爱黑白 + 少量高饱和度点缀
- **PPT 应用**：严格网格对齐；只用 1-2 种字体；颜色不超过 3 种
- **代表色**：纯白背景 + 黑色文字 + 1 个强调色（红或蓝）

### 3. Josef Müller-Brockmann — 瑞士国际主义风格之父
- **身份**：瑞士平面设计师，1914-2004
- **核心理念**：网格系统（Grid System）的集大成者
- **设计哲学**：
  - 数学网格 = 秩序 = 清晰
  - 非对称布局
  - 无衬线字体（Helvetica 诞生于这个传统）
  - 客观摄影代替插画
- **经典著作**：《Grid Systems in Graphic Design》（平面设计中的网格系统）
- **PPT 应用**：所有元素对齐到隐形网格；用比例而非感觉来放置元素
- **代表色**：白 + 黑 + 原色（红/蓝/黄）

### 4. Jony Ive（乔纳森·艾维）— Apple 设计灵魂
- **身份**：Apple 前首席设计官，iMac/iPhone/iPod 设计者
- **核心理念**：极简不是简单，而是极致的简化过程
- **设计哲学**：
  - 深受 Dieter Rams 影响
  - 功能决定形式，形式回归功能
  - 材质本身即美感（铝合金、玻璃）
  - "Simplicity is not the absence of clutter; it's the absence of complexity"
- **PPT 应用**：Apple Keynote 风格 — 全屏图片 + 大号文字；每页 3 秒内可理解；黑色背景/纯色背景
- **代表色**：黑 + 白 + 灰（Apple Keynote 经典三色）

### 5. Michael Bierut — Pentagram 合伙人
- **身份**：Vignelli 门徒，Pentagram 纽约合伙人
- **核心理念**：让设计为内容服务，而非自我表演
- **设计哲学**：
  - 欧洲形式主义 + 美国实用主义
  - "79 Short Essays on Design" 作者
  - 品牌标志性的极简表达（Mastercard、Slack）
- **PPT 应用**：标题即内容；用排版层级引导视线；装饰性元素为零
- **代表色**：根据客户定制，但始终克制（通常 2 色）

### 6. Stefan Sagmeister — 情感极简主义
- **身份**：奥地利裔美国设计师，Sagmeister & Walsh 创始人
- **核心理念**：极简不等于冷冰冰，设计应该让人感到快乐
- **设计哲学**：
  - 手工质感 + 数字精确
  - 字体即图形
  - 实验性排版
- **PPT 应用**：可以有一两个"哇"的页面（大字排版），其余保持克制
- **代表色**：白 + 黑 + 高对比度点缀

## 二、简约设计 6 大核心原则

1. **留白（Negative Space）** — 最强大的设计元素。元素之间的空间比元素本身更重要。
2. **排版层级（Typography Hierarchy）** — 用字号、字重、颜色建立信息优先级，不需要边框和装饰。
3. **限制色彩（Limited Palette）** — 不超过 3 种颜色（背景 + 正文 + 强调色）。
4. **网格系统（Grid System）** — 所有元素对齐到隐形网格，创造秩序感。
5. **一张图一个信息（One Idea Per Slide）** — 每页只传达一个核心观点。
6. **一致性（Consistency）** — 字体、颜色、间距从头到尾保持一致。

## 三、简约配色方案（PPT 适用）

### 方案 A：经典黑白（Rams/Vignelli 风格）
```
背景：#FFFFFF（纯白）
正文：#1A1A1A（近黑）
辅助：#666666（中灰）
强调：#E63946（正红）或 #2B6CB0（深蓝）
```

### 方案 B：Apple Keynote 风格（Ive 风格）
```
背景：#000000（纯黑）
正文：#FFFFFF（纯白）
辅助：#999999（浅灰）
强调：#007AFF（Apple 蓝）或 #FF9500（Apple 橙）
```

### 方案 C：温暖极简（日式/无印良品风格）
```
背景：#F5F0EB（暖白/米色）
正文：#2C2C2C（深炭）
辅助：#8B8680（暖灰）
强调：#C45B28（赤陶色）或 #3D5A80（灰蓝）
```

### 方案 D：瑞士国际风格
```
背景：#FFFFFF（纯白）
正文：#000000（纯黑）
辅助：#B0B0B0（银灰）
强调：#FF0000（红）或 #0055A4（蓝）
```

### 方案 E：高端商务
```
背景：#FAFAFA（极浅灰）
正文：#1A1A1A（近黑）
辅助：#4A4A4A（深灰）
强调：#C8A97E（金色）或 #2D3748（深蓝灰）
```

## 四、简约 PPT 字体推荐

### 无衬线（现代、干净）
- **Helvetica / Arial** — Vignelli 最爱，万能选择
- **Inter** — 现代 UI 字体，屏幕可读性极佳
- **SF Pro** — Apple 系统字体，精致
- **Noto Sans SC** — 中文场景最佳无衬线

### 衬线（优雅、权威）
- **Garamond** — 经典衬线，Vignelli 推荐
- **Source Serif Pro** — 开源，屏幕友好
- **Noto Serif SC** — 中文衬线

### 规则
- 标题和正文最多用 2 种字体
- 标题用粗体（Bold/Black），正文用常规（Regular）
- 中文字体和英文字体搭配时，选择同风格（如 Inter + Noto Sans SC）

## 五、简约 PPT 布局模板

### 布局 1：全屏大字（Sagmeister 风格）
- 全屏纯色背景
- 居中大号文字（60-80pt）
- 适合：章节分隔页、金句页

### 布局 2：左文右空（Vignelli 风格）
- 左 1/3 文字，右 2/3 留白
- 适合：要点说明、引用

### 布局 3：上图下字（Apple Keynote 风格）
- 上方 70% 全幅图片，下方 30% 文字
- 适合：产品展示、数据可视化

### 布局 4：中心聚焦（Rams 风格）
- 中心一个元素（数字/图标/短句）
- 大面积留白
- 适合：关键数据、核心观点

### 布局 5：两栏对比
- 左右各 50%，严格对齐
- 适合：前后对比、优劣势分析
