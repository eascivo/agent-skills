# Fusion Dark Style — 融合暗色风格

融合影视飓风（电影感暗色调）、半佛仙人（大字报冲击力）和何同学（干净清爽知识分享）的视觉风格。

## 核心理念

- **电影感底色** + **数据可视化** + **大字冲击** + **干净排版**
- 比纯黑更有层次的深空黑，多色系统但严格分配角色
- 不对称布局，左对齐为主，留白不等于空白

## 配色系统

| 角色 | 色值 | 用途 |
|------|------|------|
| 底色 | `#0A0E17` Deep Space | 全局背景（比纯黑更有层次） |
| 卡片暗 | `#111827` | 次级背景 |
| 卡片 | `#1A2035` | 卡片/容器背景 |
| 正文 | `#F1F5F9` | 主文字（暖白，非纯白） |
| 次要 | `#94A3B8` | 副标题/描述 |
| 暗灰 | `#475569` | 页码/脚注 |
| 电光蓝 | `#00D4FF` | 主强调色（科技/数据） |
| 珊瑚红 | `#FF4757` | 冲击色（大字报/警示） |
| 暖金 | `#FFC312` | 数据高亮/重点数字 |
| 霓虹绿 | `#00E676` | 增长/正面指标 |
| 鲜紫 | `#A855F7` | 第四维度/补充色 |
| 分隔线 | `#1E293B` | 细线/分隔 |

**颜色使用规则：**
- 每页最多 2-3 种强调色，不要彩虹
- 珊瑚红只用于需要视觉冲击的关键词（半佛式大字报）
- 电光蓝用于科技感/数据元素
- 暖金用于需要突出的数字

## 字体

| 元素 | 字号 | 字重 | 颜色 |
|------|------|------|------|
| 封面标题 | 72-80pt | Bold | `#F1F5F9` |
| 页面标题 | 32pt | Bold | `#F1F5F9` |
| 大字报陈述 | 56pt | Bold | `#F1F5F9` + 强调色 |
| 冲击数字 | 60-140pt | Bold | 强调色 |

字体族：`Helvetica Neue`（macOS 原生，无需安装）

## 布局模式

### 1. 电影感封面
- 左对齐超大标题（80pt），底部有 overlay 信息条
- 右上角装饰性半透明圆形（transparency 88-93%）
- 左上角 L 形角标（电光蓝细线）
- 英文副标题用 `charSpacing: 4` 字距

### 2. 大字报冲击页（半佛式）
- 两行大字，核心短语用珊瑚红高亮
- 底部数据条：灰色分隔线 + 彩色数据点
- 数据格式：灰色描述 + 彩色数字，用 `·` 分隔

### 3. 三栏驱动卡片
- 深色卡片（`#1A2035`）+ 顶部彩色细线（0.05 高度）
- 编号（28pt 彩色）+ 标题 + 描述
- 底部半透明 badge（transparency 85%）

### 4. 数据条形图页（何同学式）
- 左侧名称（13pt bold）+ 水平进度条 + 平台说明
- 进度条：背景 `#1E293B` + 填充色（transparency 60%）
- 右侧显示用户量（彩色）和收入范围（灰色）

### 5. 大数字冲击页
- 居中超大数字（100-140pt 彩色）
- 下方标签（20pt 白色）
- 底部 3 个辅助统计（彩色数字 + 灰色说明）

### 6. 变现路径（百分比列表）
- 左侧百分比（22pt 彩色 bold）
- 中间名称 + 详情
- 右侧比例条形图

### 7. 四象限矩阵
- 2×2 卡片布局，每张卡片有独特暗色底
- 左侧彩色竖条强调（0.05 宽度）
- 图标 + 中文标题 + 英文标签 + 描述

### 8. 时间线路径
- 水平细线 + 彩色圆点
- 上方：Phase 标签 + 大标题 + 时间段
- 下方：描述文字

### 9. 指标卡片
- 4 列深色卡片
- 超大数值（36pt）+ 单位说明 + 分隔线 + 标签 + 描述

### 10. 电影感收尾
- 两行居中大字，第二行用强调色
- 细分隔线 + 英文标语（`charSpacing: 3`）+ 中文引言

## 装饰元素

- **L 形角标**：左上角，0.4 长度，1.5pt 宽，电光蓝
- **半透明圆形**：右上角，2-3 个，transparency 88-93%
- **页面编号**：右下角，9pt 暗灰，`charSpacing: 2`，01 格式
- **信息条**：底部半透明 overlay（transparency 40%）

## 避免事项

- ❌ 不要在单页使用超过 3 种强调色
- ❌ 不要用纯黑 `#000000`（缺乏层次感）
- ❌ 不要用纯白 `#FFFFFF` 作为正文（太刺眼，用 `#F1F5F9`）
- ❌ 不要居中大段正文（左对齐为主）
- ❌ 不要用 emoji 作图标（用 Unicode 几何符号 ▶ ★ ◆ ● 或纯色圆形）
